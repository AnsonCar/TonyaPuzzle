# font  字體
- 菜單標題
菜單標題字體大小:最小取 16px，最大取 20px。
- 正文標題
正文標題字體大小:最小取 24px，最大取 32px。
- 正文的字體
一般情況下，正文字體大小以 14px 為準，特殊情況下可以加粗或取 16px 大小的字體。
所有字體樣式顏色組合需要盡量通過 3:1 的 WCAG AA 標準。
